Research Problem

Current AI text detection systems exhibit critical failures when applied to short-form content and paraphrased text. Most existing approaches depend on long-form documents (500+ tokens) and fail when text length falls below 200 tokens, a range that encompasses social media posts, comments, email snippets, and other ubiquitous forms of digital communication.

Furthermore, simple paraphrasing attacks using tools like QuillBot or DIPPER can reduce detector accuracy from 90% to near-random performance. The computational expense of neural-based detectors and their lack of interpretability present additional barriers to practical deployment.

Research Gap

No existing work systematically addresses the intersection of short text length and paraphrasing robustness using lightweight statistical methods.

Analysis & Key Insights

Lexical Diversity Patterns

Human-authored text demonstrates significantly higher vocabulary richness across all Type-Token Ratio variants (standard, root, logarithmic). The most discriminative feature, Lexical Log TTR, achieves an effect size of d = 1.76 with p = 9.83×10⁻⁶⁴.

This pattern persists even under paraphrasing attacks, as synonym substitution cannot fundamentally alter the underlying vocabulary distribution patterns that characterize human linguistic creativity versus AI model constraints.

Practical Implication

TTR-based features provide robust first-line detection signal requiring minimal computational resources, suitable for real-time screening applications.

Information Compression Signatures

AI-generated text exhibits higher compression ratios (lower entropy), indicating greater structural predictability. The compression ratio feature achieves d = 1.57 with p = 3.20×10⁻⁶⁰, ranking second among all tested metrics.

Language models optimize for high-probability token sequences during generation, resulting in text that compresses more efficiently than human writing, which contains more unpredictable elements, false starts, and stylistic variations.

Theoretical Significance

Compression-based features capture fundamental differences in generative processes, making them resistant to surface-level paraphrasing manipulations.

Punctuation Pattern Analysis

Question mark usage demonstrates unexpectedly strong discrimination (d = 1.39, p = 1.66×10⁻⁵⁶). Human authors employ interrogative structures more frequently, while AI models favor declarative statements even when conversational tone is prompted.

This finding suggests that current language models have systematic biases in rhetorical structure that persist across different prompting strategies and model architectures.

Detection Application

Punctuation features provide orthogonal signal to lexical metrics, enabling ensemble approaches that combine multiple feature categories for improved robustness.

Cross-Model Generalization

Features demonstrate consistent discrimination across multiple language model families (GPT-series, Bard, DeepSeek), suggesting capture of fundamental AI generation characteristics rather than model-specific artifacts.

This cross-model stability indicates potential for long-term detector viability even as language models continue to evolve, addressing a critical limitation of current neural detection approaches that require retraining for each new model version.

Future Robustness

Model-agnostic features provide foundation for sustainable detection systems that maintain effectiveness across rapid model development cycles.

Feature Correlation Structure

Correlation analysis reveals distinct feature clusters measuring related linguistic properties. High correlation (r > 0.9) between TTR variants confirms measurement of similar lexical diversity constructs through different normalizations.

Negative correlations between readability metrics and compression features indicate complementary information capture. This orthogonality supports ensemble approaches combining features from different categories.

Feature clustering enables dimensionality reduction strategies for deployment scenarios requiring minimal computational overhead, as redundant features can be pruned without substantial accuracy loss.

Machine Learning Implication

Low multicollinearity among top features suggests stable coefficient estimation in linear models and reduced overfitting risk in tree-based ensemble methods.

Research Gap Analysis

Current State of AI Text Detection

Existing detection systems predominantly focus on long-form documents where statistical anomalies accumulate sufficiently for reliable identification. Performance degrades dramatically when text length falls below 200 tokens, with most systems approaching random-chance accuracy at 50-100 token ranges.

Paraphrasing Vulnerability

Simple paraphrasing tools like QuillBot can reduce neural detector accuracy from 90%+ to 50-60%. More sophisticated attacks using models like DIPPER can evade detection entirely while preserving semantic content. No published work systematically addresses both short text and paraphrasing robustness simultaneously.

Our Approach

This research specifically targets the intersection of these challenges through lightweight statistical features designed for stability under lexical manipulation. The 37-feature framework captures multiple orthogonal aspects of text generation, providing redundancy against adversarial attacks.

Research Contributions

1. Comprehensive Feature Framework

Development and validation of 37-feature stylometric framework spanning six linguistic categories. Systematic evaluation identified 22 features with statistical significance (p < 0.05) and large effect sizes (d > 0.8 for 15 features).

2. Paraphrasing Robustness Demonstration

Empirical validation across three paraphrasing intensities (original, 30% substitution, 50% substitution with reordering). Top features maintain discriminative power with less than 10% accuracy degradation under heavy paraphrasing.

3. Cross-Model Generalization

Demonstrated effectiveness across multiple language model families (GPT-series, Bard, DeepSeek), suggesting capture of fundamental generation characteristics rather than model-specific artifacts.

4. Computational Efficiency

Lightweight feature extraction requiring no GPU infrastructure. Complete analysis of 600 samples completes in under 30 minutes on standard desktop hardware. Enables real-time screening applications impossible with neural detectors.

Practical Impact & Applications

Academic Integrity Systems

Short-form detection enables monitoring of discussion posts, comments, and assignment submissions where length constraints previously prevented reliable detection. Paraphrasing robustness addresses primary evasion strategy used by students.

Social Media Content Authenticity

Twitter/X posts, Reddit comments, and Facebook updates typically contain 20-200 tokens. This research enables detection at scales matching actual social media content distribution, addressing concerns about coordinated inauthentic behavior and misinformation campaigns.

Email and Communication Screening

Lightweight detection suitable for real-time integration into email systems, customer support platforms, and messaging applications. Computational efficiency enables deployment without specialized infrastructure.

Content Authenticity Verification

Paraphrasing Methodology

Systematic synonym replacement using curated dictionary of 50+ common word substitutions. Light paraphrasing applies 30% replacement probability per eligible token. Heavy paraphrasing increases to 50% and additionally shuffles sentence order in multi-sentence passages.

This approach simulates realistic evasion attempts while maintaining experimental control. More sophisticated attacks using neural paraphrasers planned for Phase 4 adversarial evaluation.

Statistical Methodology

Hypothesis Testing Framework

Mann-Whitney U test employed for all feature comparisons due to non-normal distributions in stylometric data. This non-parametric test makes no assumptions about underlying distribution shape, providing robust statistical inference.

Multiple Comparison Correction

Bonferroni correction applied to control family-wise error rate across 37 simultaneous tests. Adjusted significance threshold: α = 0.05/37 = 0.00135. Despite this conservative correction, 22 features achieve significance.

Effect Size Interpretation

Cohen's d calculated using pooled standard deviation. Conventional interpretation: small (d ≈ 0.2), medium (d ≈ 0.5), large (d ≈ 0.8). Top features achieve d > 1.2, considered "very large" effects rarely observed in behavioral research.

Sample Size Justification

Power analysis conducted using G*Power software. For detecting medium effects (d = 0.5) with α = 0.05 and power = 0.80, minimum required n = 64 per group. Our sample sizes (300 per group) provide >99% power for medium effects and adequate power for small effects (d = 0.2).
